# RESOURCE-HUB-archive

**RESOURCE-HUB 资源备份存档**  
本仓库用于存放 **RESOURCE-HUB** 的各版本/各资源备份下载地址，以防主站下载链接失效时无法获取所需资源。

## 📦 用途

- 备份 RESOURCE-HUB 中重要的资源文件下载链接  
- 提供备用下载地址，提高资源可用性  
- 记录资源版本更新历史，便于回溯
